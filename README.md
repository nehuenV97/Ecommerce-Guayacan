## Instalacion

Clonar/Descargar el repositorio.

Instalar el proyecto con npm:
```bash
  npm install
```

Iniciamos el proyecto en un servidor:
```bash
  npm start
```

Visitamos el proyecto en el sevidor: 
```bash
  http://localhost:3000/
```
